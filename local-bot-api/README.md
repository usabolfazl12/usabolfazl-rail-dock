# 🚀 سرور محلی Telegram Bot API (عبور از محدودیت ۵۰ مگابایت)

با این روش، سرور رسمی `tdlib/telegram-bot-api` را با Docker خودت اجرا می‌کنی و
ربات به‌جای Bot API عمومی به آن وصل می‌شود. نتیجه: سقف آپلود از **۵۰MB** به
**۲GB** می‌رسد.

> بر اساس مقاله‌ی Joy Biswas در dev.to

---

## پیش‌نیاز
- **Docker** و **Docker Compose** نصب باشد
- یک سرور/کامپیوتر که همیشه روشن است (VPS یا PC خودت). ⚠️ روی Replit عملی نیست.

---

## مرحله ۱ — گرفتن API_ID و API_HASH
1. برو به: https://my.telegram.org/auth
2. با شماره تلفن تلگرامت وارد شو
3. **API development tools** → یک app بساز
4. مقدار `api_id` و `api_hash` را بردار

## مرحله ۲ — ساخت فایل .env
فایل `.env.example` را کپی کن به `.env` و مقادیر واقعی را بگذار:

```bash
cp .env.example .env
# سپس .env را ویرایش کن:
# API_ID=12345678
# API_HASH=abcd1234...
```

## مرحله ۳ — بالا آوردن سرور
```bash
docker compose up -d --build
```
(اولین بار چند دقیقه طول می‌کشد چون سرور از سورس کامپایل می‌شود.)

## مرحله ۴ — تست کردن
```bash
curl "http://localhost:8081/bot<TOKEN>/getMe"
```
`<TOKEN>` را با توکن رباتت جایگزین کن. اگر JSON با مشخصات ربات برگشت، یعنی کار می‌کند. ✅

---

## مرحله ۵ — وصل کردن ربات به سرور محلی
کد ربات (`my bot.txt`) از قبل آماده است. فقط کافی است متغیر محیطی زیر را ست کنی:

```
TG_API_URL=http://localhost:8081
```

- اگر ربات روی همان ماشین Docker است: `http://localhost:8081`
- اگر ربات جای دیگری است: `http://<آی‌پی سرور>:8081`

ربات را ری‌استارت کن. در لاگ باید ببینی:
```
استفاده از Local Bot API Server: http://localhost:8081 (سقف 2GB)
```
و پیام‌های خطای حجم هم به‌جای «50MB» می‌نویسند «2GB». 🎉

## نکات مهم
| نکته | توضیح |
|------|-------|
| 🔄 **logout از سرور رسمی** | وقتی به سرور محلی سوییچ می‌کنی، بار اول ممکن است لازم باشد ربات از سرور رسمی logout شود. اگر خطای `logged in` دیدی، چند دقیقه صبر کن یا با متد `logOut` خارج شو. |
| 💾 **حجم دیسک** | فایل‌های دانلودی در volume به اسم `telegram-bot-api-data` ذخیره می‌شوند؛ حواست به فضای دیسک باشد. |
| 🔒 **پورت** | پورت 8081 را روی اینترنت باز نکن مگر پشت فایروال/پروکسی. |
| 📦 **سقف واقعی** | آپلود تا 2GB، دانلود فایل از تلگرام هم بدون محدودیت 20MB می‌شود. |
